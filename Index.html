B2 PRODUCTION SYSTEM

Chưa tìm thấy cấu hình Firebase hợp lệ. Làm theo 4 bước bên dưới rồi tải lại trang.

Bước 1 — Vào console.firebase.google.com, tạo project mới.

Bước 2 — Bật Authentication → Sign-in method → Email/Password.

Bước 3 — Tạo Firestore Database (chế độ Production). Nếu tab "Báo cáo tồn kho" báo lỗi yêu cầu tạo index, bấm vào đường link trong lỗi đó để Firebase tự tạo index (chỉ cần làm 1 lần).

Bước 4 — Vào Project settings → General → Your apps → Web app, copy đoạn firebaseConfig rồi dán thay vào biến firebaseConfig ở đầu thẻ <script type="module"> trong file HTML này.

Gợi ý Firestore Security Rules (dán vào tab Rules):

rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
    function signedIn() { return request.auth != null; }
    function myDoc() { return get(/databases/$(db)/documents/users/$(request.auth.uid)); }
    function isApproved() { return signedIn() && myDoc().data.status == 'approved'; }
    function isAdmin() { return isApproved() && myDoc().data.role == 'admin'; }

    match /users/{uid} {
      allow create: if signedIn() && request.auth.uid == uid;
      allow read: if signedIn();
      allow update: if isAdmin() || (signedIn() && request.auth.uid == uid && request.resource.data.role == resource.data.role);
    }
    match /items/{id} {
      allow read: if isApproved();
      allow create: if isApproved();
      allow update: if isApproved();
      allow delete: if isAdmin();
    }
    match /boms/{id} {
      allow read: if isApproved();
      allow write: if isAdmin();
    }
    match /materialReceipts/{id} {
      allow read: if isApproved();
      allow create: if isApproved();
      allow update, delete: if isAdmin();
    }
    match /productionReports/{id} {
      allow read: if isApproved();
      allow create: if isApproved();
      allow update, delete: if isAdmin();
    }
    match /stockIssues/{id} {
      allow read: if isApproved();
      allow create: if isApproved();
      allow update, delete: if isAdmin();
    }
    match /reworks/{id} {
      allow read: if isApproved();
      allow create: if isApproved();
      allow update, delete: if isAdmin();
    }
    match /stockLedger/{id} {
      allow read: if isApproved();
      allow create: if isApproved();
      allow update, delete: if isAdmin();
    }
    match /monthlyClosingSnapshots/{id} {
      allow read, write: if isApproved();
    }
    match /lots/{id} {
      allow read: if isApproved();
      allow create, update: if isApproved();
      allow delete: if isAdmin();
    }
    match /counters/{id} {
      allow read, write: if isApproved();
    }
    match /packingEvents/{id} {
      allow read: if isApproved();
      allow create: if isApproved();
      allow update, delete: if isAdmin();
    }
    match /stockIssueBatches/{id} {
      allow read: if isApproved();
      allow create: if isApproved();
      allow update, delete: if isAdmin();
    }
  }
}

Nếu bạn đã publish rules cũ trước đây, hãy vào Firestore → Rules và dán đè bộ rules mới này rồi Publish lại để phân quyền sửa/xoá giao dịch (chỉ Admin) có hiệu lực, và để tính năng "Xuất kho theo lô — quét nhiều cuộn" (collection stockIssueBatches) hoạt động.
